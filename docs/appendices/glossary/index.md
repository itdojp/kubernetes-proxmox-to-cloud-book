# 用語集

## この章の学習目標（3〜5）

- 本書で頻出する用語の定義を確認できる
- 同義語・誤解しやすい用語を区別できる

## 用語（ドラフト）

| 用語 | 定義（ドラフト） |
| --- | --- |
| Proxmox VE | 仮想化基盤（Kubernetesの外側） |
| kubeadm | Kubernetesクラスタ初期化ツール |
| containerd | コンテナランタイム（CRI実装） |
| CNI | Kubernetesネットワークのプラグイン仕様 |
| MetalLB | ベアメタル/オンプレ向けロードバランサ実装 |
| Kustomize | YAML の差分を overlay で管理する仕組み |
| Helm | Chart と values でリリースを管理する仕組み |

## まとめ

用語の意味が曖昧なまま進むと、設計判断の前提が崩れます。迷ったらここに戻って確認してください。
