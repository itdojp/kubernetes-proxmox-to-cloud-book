# コントリビューションガイド

## 対象

- 誤字脱字、技術的な不備、表現の改善提案
- 章構成、ナビゲーション、サンプル手順の改善提案

## 手順（推奨）

1. Issue を作成し、背景・期待する改善・受け入れ条件を記載してください。
2. 変更提案が実装を伴う場合は Fork し、Pull Request を作成してください。
